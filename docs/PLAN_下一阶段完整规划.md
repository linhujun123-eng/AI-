## 二、冷启动前要达到的产品标准

基于 4.17 推广策略讨论的共识，冷启动定位是 **「吉他练歌神器」**：

> 一句话：用你喜欢的歌练吉他，和弦自动标注，难的地方慢练循环。

用户打开产品时，需要感受到：
1. **开箱即练** — 有丰富的热门曲目可选，不需要自己找歌上传
2. **和弦跟练体验好** — 能看到和弦走向，能看到指法图，能跟着练
3. **工具感强** — AB 循环、变速、分轨，比 Moises 更聚焦练习
4. **可安装** — PWA 添加到主屏，像 App 一样使用
5. **记住我** — 下次回来能续练，有进步感知
